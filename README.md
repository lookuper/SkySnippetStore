# SkySnippetStore
=================

SnippetStore application implement as SPA via AngularJS

Requirenments: 
  Google Chrome v22+
  Google Account (optional)
  
Features: 
  JavaScript code syntax highlighting *.
  Upload\Download to Google Drive.
* - using Ace Editor control from http://ace.c9.io/
